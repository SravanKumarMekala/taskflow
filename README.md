# TaskFlow — Smart Task Manager 📋

A clean, responsive task management web application built with **Vanilla JavaScript**, **HTML5**, and **CSS3**. No frameworks, no dependencies — pure web fundamentals.

🔗 **Live Demo:** [https://yourusername.github.io/taskflow](https://yourusername.github.io/taskflow)

---

## 📸 Preview

> A dark-themed, responsive task manager with priority levels, filters, real-time stats, and progress tracking.

---

## ✨ Features

- ✅ **Add, complete & delete tasks** with smooth animations
- 🔴🟡🔵 **Priority levels** — High, Medium, Low with color-coded badges
- 🔍 **Filter tasks** by status (All / Pending / Completed) and priority
- 📊 **Live stats** — Total, Done, Pending counts update in real time
- 📈 **Progress bar** showing overall completion percentage
- 💾 **Persistent storage** — Tasks saved to `localStorage`, survive page refresh
- ⏰ **Live clock** in the header
- ⌨️ **Keyboard shortcut** — Press `Enter` to add a task
- 📱 **Fully responsive** — Works on mobile, tablet, and desktop
